I try to use a very simple example to demonstrate how abaqus/standard works, i.e. the algorithm used in Abaqus. Following is an example to verify the principle.

  1. A C3D8 brick is developed in Matlab. A single element 2x2x2 cm^3 is tested.

  2. Linear elastic material model is used in the element. By taking large displacement into consideration, more accurately speaking, the material model is called hypoelasticity.

  3. The updated Lagrange format is used in the algorithm.
