FEM

I try to compare my nonlinear FEM code with Abaqus/standard. Following is an example.

  1. An implicit solver and a C3D8 brick are developed in Matlab. A single element (2x2x2 cm^3) is tested with one facet fixed and the opposite facet subject to extension or compression forces

  2. Hypoelasticity is assumed in the element.

  3. The updated Lagrange format is used in the algorithm.

  4. Comparison:

From the comparison, one might find the result is different from the one given in K.J. Bathe's book or ADINA. The discreprancy stems from the different procedures to approximate/calculate stress.
