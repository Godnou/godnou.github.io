On coding a general user material subroutine

2019-04-30
FEM

As a tutorial, this post shows a simple and extensible user material subroutine (UMAT), where large deformation is considered. For simplicity, the hopyelasticiy is adopted as the material law. The robustness and correctness of the general UMAT is demonstrated in two examples: one is uniaxial extention of a single element; the other is triaxial compression of a single element.

Caption: (a) uniaxial extension test of a single element (Force v.s. extention). (b) triaxial compression test of a single element (Volume strain v.s. compression). Both examples show the consistent results with one obtained by the built-in subroutine.

The source code can be found here: Hypoelasticity

Detailed comments are added in the source code for explanation. I hope every beginner can learn something from the source code. The framework designed in the source code can be easily applied to other material laws, such as hyperelasticity, plasticity and viscoelasticity.
